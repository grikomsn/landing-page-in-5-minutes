# 😪 Create now.json

```json
{
  "name": "landing-page-in-5-minutes",
  "version": 2,
  "alias": ["landing-page-in-5-minutes"],
  "builds": [
    {
      "src": "package.json",
      "use": "@now/static-build",
      "config": { "distDir": "public" }
    }
  ],
  "public": true,
  "regions": ["all"]
}
```

---

# ✏️ Modify package.json

```json
{
  "scripts": {
    ...
    "now-build": "yarn build"
  }
}
```

Deployment will fail if there is no `now-build` to execute
